    BitCountAttribute

    Tells weaver how many bits to sue for field

    Only works with interager fields (byte, int, ulong, enums etc)

    NOTE: bits are truncated when using this, so signed values will lose their sign. Use ZigZagEncodeAttribute as well if value might be negative

    Also See: Bit Packing Documentation
